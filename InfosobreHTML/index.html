Conceptos Basicos



Estructura

-HTML5 introdujo nuevos elementos de base estructural, los cuales son:

<header>
Es usado para indentificar el inicio de la página, artículo, sección, u otro segmento de una página.
<nav>
Identifica una sección principal, referida a la navegación dentro de esta, incluyendo links a cada parte de la página.
<article>
Es usado para idenfiticar una sección independiente, y con contenido autónomo.
<section>
Es usado para identificar un grupo de contendio agrupado por una misma temática.
<aside>
Su contendio, como por ejemplo las barras laterales, inserciones, o breves explicaciónes, esta relacionado al contenido cercano, siendo lo de aside, un complemento para este.
<footer>
Identifica el final de página, artículo, sección, u otro segmento de una página



Interlineado

Para dar un salto de linea, se utiliza el elemento <br>.
Támbien se puede utilizar <hr> para generar una linea divisioria




Hiperlinks

Los enlaces que apuntan a otras páginas del mismo sitio web tendrán una ruta relativa, que no incluye el dominio (.com, .org, .edu, etc.) en el valor del atributo href.
La vinculación a otros sitios web fuera del actual requiere una ruta absoluta, donde el valor del atributo href debe incluir el dominio completo.
Ejemplos:

  • Relative Path → <a href="about.html">About</a>
  • Absolute Path → <a href="http://www.google.com/">Google</a>

Para crear un enlace de correo electrónico, el valor del atributo href debe comenzar con mailto: seguido de la dirección de correo electrónico a la que se debe enviar el correo electrónico.
Ejemplo: <a href="mailto:shay@awesome.com?subject=Reaching%20Out&body=How%20are%20you"> Envíeme un correo electrónico </a>.


Para activar la acción de abrir un enlace en una nueva ventana, use el atributo de destino con un valor de _blank. El atributo de destino determina exactamente dónde estará el enlace mostrado, y el valor _blank especifica una nueva ventana.
Ejemplo: <a href="http://shayhowe.com/" target="_blank"> Shay Howe </a>
