Tablas

Las tablas se arman con <table>. Y una tabla como minimo incluira <tr> (table row) y <td> (table data). Ademas puede incluir un table header, con <th>.
Los headers utilizan "scope" para marcar a que contenido se relacionan (col, row, colgroup, and rowgroup). Se puede usar el elemento in-line "border" para darle borde a las celdas.
Además, se puede utilizar rowspan y colspan para combinar celdas, como elemento in-line de <td>.
