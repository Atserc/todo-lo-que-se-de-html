Títulos y Párrafos

El título principal de la página, o de la sección debería ser marcado con un elemento <h1>, y los títulos subsecuentes, deberían usar <h2>, <h3>, <h4>, <h5>, and <h6> a medida que son necesarios.
Usar estos titulos, ayuda a desarmar el contenido y establecer jerarquia, y son identificadores clave para los usuarios que lean la página También ayudan a los buscadores a clasificar y determinar el contenido de la página.

Los titulos son usualmente seguidos por párrafos que los complementan. Estos párrafos se definen usando el <p> elemento block-level. Los párrafos pueden uno después del otro, agregando información a la página según sea deseado.


Negrita y Cursiva

Para hacer el texto en negrita y darle fuerte importancia, se usa el elemento inline <strong>. Además de este elemento, se puede usar <b>.
Para la italica, se utiliza el elemento inline <em>. Este elemento es usado semanticamente para darle un énfasis en el texto, y es la opcion mas popular para realizar esto. La otra opción, es el elemento <i>, que es semanticalmente usado para transmitir texto en una voz o tono alternativo, generalemente usado también para citaciones.
